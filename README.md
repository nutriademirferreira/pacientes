# Nutri Ademir Ferreira — Consultório

Repositório central. Todos os documentos de pacientes, publicados via GitHub Pages.

Site: https://nutriademirferreira.github.io/pacientes/

## Estrutura

```
nutri-consultorio/
├── pacientes/
│   └── nome-sobrenome/
│       ├── protocolo/          ← HTMLs do plano alimentar
│       ├── avaliacao-fisica/   ← PDFs de composição corporal
│       ├── exames/             ← resultados laboratoriais
│       ├── cartas-medicas/     ← encaminhamentos ao médico
│       └── devolutivas/        ← protocolos de suplementação
├── _config/                    ← configurações PWA
├── _templates/                 ← modelos reutilizáveis
├── _scripts/                   ← automações
└── README.md
```
